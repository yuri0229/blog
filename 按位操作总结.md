## & 按位与
### 1、将指定位清零
> 例1：将a的A5-A3位置成零，用b与之进行&操作，得到c即是清零后的结果

		A5	A4	A3	A2	A1	A0

	a =>	1	1	1	1	1	1

	&

	b =>	0	0	0	1	1	1

	=

	c =>	0	0	0	1	1	1

### 2、取余数
> 例2：求a % b，其中b是2的n次方

a % b = a & (b - 1)

设a = 63，b = 8，那么 63 % 8 = 63 & 7 = 7，正好和例1相同

### 3、判断奇偶性
> 例3：判断a是奇数还是偶数

a & 1 == 1 为奇数

a & 1 == 0 为偶数

## | 按位或
### 1、按位赋值
> 例：将a赋值给b（先将b清零）, b = b | a

	a	=>	1	0	1	1	1	0

	|

	b	=>	0	0	0	0	0	0

	=
			1	0	1	1	1	0


## ^ 按拉异或
### 1、将指定位取反
> 例：将a的A5-A3位置取反得到c，c = a ^ b

		A5	A4	A3	A2	A1	A0
		
	a =>	1	0	0	1	0	0

	&

	b =>	1	1	1	0	0	0

	=

	c =>	0	1	1	1	0	0


### 2、不引入第三个变量情况下将两个变量值互换
> 例：

a = 0 1 0

b = 1 1 0

a = a ^ b = 1 0 0

b = a ^ b = 0 1 0

a = a ^ b = 1 1 0
